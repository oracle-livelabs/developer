# Oracle RAG Agents: Zero to Hero

## Introduction

In this workshop, you will build a research assistant on Oracle AI Database 26ai from the ground up. You will store research papers in Oracle, index them for text, vector, and graph retrieval, ground answers with Retrieval-Augmented Generation (RAG), and then extend the flow into tool-using agents with persistent memory in the same database.

This is a full-stack Oracle AI Database workshop. You will stand up the database locally, prepare the corpus, compare retrieval strategies on one dataset, connect the results to OpenAI models, and finish with an agentic workflow that stores its own history in Oracle.

Estimated Workshop Time: 1 hour 45 minutes

### Objectives

In this workshop, you will:

- prepare a Jupyter environment for an AI database workflow
- install and connect to Oracle AI Database 26ai locally
- load and embed a research-paper dataset
- build text, vector, hybrid, and graph retrieval in Oracle
- run an end-to-end RAG pipeline with OpenAI
- build multi-tool agents with chat history and persistent Oracle-backed memory

### Prerequisites

This workshop assumes you have:

- a machine where you can run Docker and JupyterLab
- Python 3.10 or later
- internet access for Python packages, Hugging Face datasets, and OpenAI API calls
- an OpenAI API key
- enough local memory and disk to run Oracle Database Free, Sentence Transformers, and the required notebook packages together
